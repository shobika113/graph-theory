This project focuses on racing track optimization using Graph Theory–based path analysis techniques. The system models racing circuits as a weighted graph structure, where track checkpoints, turns, and key segments are represented as vertices (nodes) and the track segments connecting them are represented as edges. Each edge is assigned a weight parameter such as distance, estimated travel time, or energy consumption, enabling efficient computation of optimal racing paths.

To determine the most efficient route across the circuit, the system implements the Dijkstra Shortest Path Algorithm, a greedy algorithm used to solve single-source shortest path problems in weighted graphs. The algorithm iteratively evaluates edge weights and updates the minimum cost from the starting node to all other nodes using a priority-based traversal mechanism, ensuring the optimal path is identified. In this context, edges represent track segments between corners, straights, and checkpoints, allowing the algorithm to compute the most efficient traversal across the circuit.

This graph-based optimization approach was applied to analyze racing circuits such as the Lusail International Circuit and our testing track Coastt Kartmania. The system helped evaluate optimal track navigation strategies during vehicle testing and simulation.

This work was developed as part of Team Renew’s project for the Shell Eco-marathon 2026, where the team secured 3rd place in the Hydrogen Category. The algorithm contributed to track analysis and energy-efficient racing strategy planning, supporting the vehicle’s performance during the competition.
